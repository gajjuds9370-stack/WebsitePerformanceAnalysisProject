🌐 Website Performance Analysis
📌 Project Overview

This project focuses on analyzing website performance data to uncover user behavior patterns, traffic trends, and engagement insights. Using a dataset containing 3000+ records, I performed Exploratory Data Analysis (EDA) and created visualizations to answer real-world business questions related to website traffic and marketing channel effectiveness.

🎯 Objectives

The main goal of this project is to help businesses understand:

Website traffic trends over time
User engagement behavior
Performance of different marketing channels
Peak traffic hours
Correlation between traffic and engagement metrics
📊 Dataset Features
Column	Description
Channel Group	Traffic acquisition source
Date	Session date
Hour	Session hour
Users	Number of users
Sessions	Number of sessions
Engaged Sessions	Sessions with meaningful interaction
Average Engagement Time per Session	Average user engagement duration
Engaged Sessions per User	Engagement metric per user
Events per Session	Average events generated per session
Engagement Rate	Percentage of engaged sessions
Event Count	Total events generated
Non-Engaged	Non-engaged sessions
🔍 Business Questions Solved
1. Traffic Trend Analysis
How do users and sessions change over time?
Are there any seasonal or daily traffic patterns?
2. Marketing Channel Performance
Which channel brings the highest number of users?
Which channel generates the most sessions?
3. Engagement Analysis
Which channel has the highest average engagement time?
How does engagement rate vary across channels?
4. Session Quality Analysis
Which channels drive the most engaged sessions?
Which channels have the highest non-engaged sessions?
5. Peak Traffic Hours
At what hours does each channel drive maximum traffic?
6. Correlation Analysis
Is there a relationship between website traffic and engagement rate?
